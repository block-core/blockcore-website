Chain Setup

We at Blockcore believe in making blockchain technology easier and more approachable for every developer, so part of this guides us in how we build our technologies. We attempt to make our technology easily customizeable through setup and configuration, as oppose to code changes.

We have a seperate web site where we publish chain setup information for the different chains. While this is officially hosted by Blockcore, it is the individual teams that are responsible for maintaining these setup definitions.
