If you are building and running a blockchain based on Blockcore, please let us know and we can get you listed below.

Reference Nodes

The Blockcore team provides reference implementations for various blockchains. These are unsupported node software, use with caution. You can look at these to get inspiration and examples on building your own customized blockchain.

https://github.com/block-core/blockcore-nodes

Chain Setup

We at Blockcore believe in making blockchain technology easier and more approachable for every developer, so part of this guides us in how we build our technologies. We attempt to make our technology easily customizeable through setup and configuration, as oppose to code changes.

We have a seperate web site where we publish chain setup information for the different chains. While this is officially hosted by Blockcore, it is the individual teams that are responsible for maintaining these setup definitions.

https://chains.blockcore.net

Hosted Services

While anyone are free to fork and modify our software, we also host our own services running some of the blockchains that are implemented on Blockcore. These are hosted with no SLA (Service Level Agreement), and currently for free. If you'd like us to continue providing this service to the community, consider becoming a sponsor of Blockcore.
