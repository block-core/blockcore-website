Platform

The platform is our ecosystem of various software and services that aligns with the vision of building decentralized software.

We have blockchain Node software that is used by multiple blockchains. We have an Indexer (blockchain API) that builds a database of history, including tokens, NFTs, transactions and more. We have the Explorer that renders content from the Indexer We have the Wallet, which is a non-custodial wallet where the user is in control of their own keys. We have the DID Server for hosting and publishing decentralized identifiers. We have the Vault for storing and sharing your whole life of data in a private and secure manner. And even more such as Tipbot and decentralized DNS software.

Node

The node software is a full node software that runs the blockchain network. It is compatible with Bitcoin and utilized by other blockchains at the primary full node.

To build your own blockchain based on Blockcore you have two options; perform a fork of our primary source code, this is preferred if you need to radically modify the blockchain functionality.

The other way is to utilize our blockchain template, which is generated and relies on published on NuGet. This enables you to quickly and easily launch your own blockchain, and you will easily be able to get latest updates without messing with source code.

To simplify using our tools, we have made instructions and generators available here.

Source repo for template-based node: https://github.com/block-core/blockcore-node (Preferred)

Source repo for fork-based node: https://github.com/block-core/blockcore

Explorer

Together with the Blockcore Indexer (below), we have made the Blockcore Explorer. If you have a normal Blockcore based blockchain, it is super easy to add support for your custom chain and deploy your blockchain explorer.

The explorer has sections for Network and Insight, which shows statistics on the network and supply (total supply, circulating supply, rewards) and economy information on the blockchain.

Code: https://github.com/block-core/blockcore-explorer

Indexer

Being able to look up blocks, transactions and addresses is an important part of any blockchain. It is very easy to add your own custom blockchain to the explorer.

Code: https://github.com/block-core/blockcore-indexer

Wallet

Blockcore Wallet is a Blockcore, Stratis and Cirrus Non-Custodial HD wallet in your browser for Coins, Tokens, Identities (DID), NFTs and more.

Extension: https://chrome.google.com/webstore/detail/blockcore-wallet/peigonhbenoefaeplkpalmafieegnapj

Code: https://github.com/block-core/blockcore-wallet

Vault

Decentralized Web Node (DWN) for Web5, includes Verifiable Data Registry (VDR) for Decentralized Identifiers (DID) and Verifiable Credentials (VC) storage.

Code: https://github.com/block-core/blockcore-vault

Tipbot

Blockcore Tipbot is a generic tipping bot for Discord (future support for Twitch and Twitter is being added) that supports all Blockcore blockchains. It is easy to configure and use, and allows communities to easily tip based on community work and contributions.

Code: https://github.com/block-core/blockcore-tipbot

Identifiers

Blockcore is committed to supporting the ecosystem of Decentralized Identifiers (DIDs) and the DID method for Blockcore is "did:is:".

DID Method: https://did.is

Blockcore DID Server: https://github.com/block-core/blockcore-did-server

Specification: https://github.com/block-core/blockcore-did-method

Status

Blockcore Status is an dashboard that monitors servers and services, source code repository, release versions and more. It's a single source for getting an complete overview of your Blockcore-based blockchains.

Code: https://github.com/block-core/blockcore-status
