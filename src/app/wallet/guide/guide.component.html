Wallet Guide

Every single feature demonstrated below, this is a quick way to learn how to use our wallet and experience what features exists.

Installation of Extension

You can install and use the Blockcore Wallet either as an browser extension, or directly in a browser tab.

Some features are only available when installed as a browser extension.

Install Extension

Installation of Web App

You can install the Web Wallet using your web browser, giving the wallet a dedicated icon on your start menu.


Open Web Wallet

Create Wallet

To get started, go through the "Create New Wallet" guide and select which type of accounts you want to use.

Edit Wallet and Account

You can at any time edit your wallets and accounts. You can change the name and icons.

You can also remove individual accounts and complete wallets and all accounts in it.

Create Account

You can add multiple accounts to your wallets, even after the initial setup.

Go to the menu on the top right, and choose "Add Account". Fill out what network and other details you need.

Receive Payment

You can receive payment either by using your next receive address, or by creating a payment request.

Every time you receive payment, you get a new receive address. This is for your privacy.

Address and Payment Requests can be shared as text by messenger, e-mail, or by QR codes that can be scanned by mobile phones.

Send Payment

You can send payment very easily and quickly. If you receive a payment request, you can paste that and all fields will be auto-filled.

You can enter a personal memo that is stored locally in your account history.

Address Book

You can add your contacts and their address, for different networks, to quickly send multiple payments to the same contact.

When you paste contact address, we'll attempt to automatically recognice what network it belongs to.

Settings

You can access settings from the left menu. It allows you to customize the wallet according to your preferences.

More language support is added all the time, if your native language is not yet supported, please let us know.

Protocol Handler

First action in settings is Protocol Handler. This allows you to register the wallet to handle links that have the "web+pay" prefix.

When this is enabled, when you click links that has a payment request in it, the wallet will automatically open.

Network Status

The network status gives you insight to the current available servers for the accounts you have in your wallet.

The buttons allows you to select a different than the automatically selected server.

Change Password

You can change the password that is currently used to securely save your wallet on your device.

The password should be as strong as possible. If your device is lost and unlocked, an attacker can run computers to guess your password and gain access to your wallet.

Reveal Recovery Phrase

Your secret recovery phrase, should at all times be kept secret and hidden from others.

If you share this with anyone, they can take control of everything.

If you lost your recovery phrase or need access to it, you can reveal it with your password in settings.
