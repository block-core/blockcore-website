Wallet Guide

Every single feature demonstrated below, this is a quick way to learn how to use our wallet and experience what features exists.

Installation of Extension

You can install and use the Blockcore Wallet either as an browser extension, or directly in a browser tab.

Some features are only available when installed as a browser extension.

Install Extension

Installation of Web App

You can install the Web Wallet using your web browser, giving the wallet a dedicated icon on your start menu.


Open Web Wallet

Create Wallet

To get started, go through the "Create New Wallet" guide and select which type of accounts you want to use.

Edit Wallet and Account

You can at any time edit your wallets and accounts. You can change the name and icons.

You can also remove individual accounts and complete wallets and all accounts in it.
